# Security Policy

## Supported Versions

We support security fixes for the latest minor release and the active production branch.

| Version | Supported |
| ------- | --------- |
| Latest  | yes       |
| Older   | no        |

## Reporting a Vulnerability

Please report security issues privately.

- Email: security@example.com
- Expected response time: 2 business days
- Triage target: 5 business days
- Fix target: 30 days (severity dependent)

If you believe you have found a critical vulnerability, please include:
- A clear description of the issue
- Reproduction steps or a proof-of-concept
- Potential impact and attack surface
- Any suggested mitigations

We will acknowledge receipt, investigate, and coordinate a fix and disclosure timeline.
